# タスク: i18n翻訳レビュー

## 概要
各ワーカーが行った国際化翻訳作業をレビューする。

## レビュー対象ブランチ
- `feat/tournament-i18n` - 大会ページ翻訳
- `feat/series-i18n` - シリーズページ翻訳
- `feat/auth-i18n` - 認証ページ翻訳
- `feat/mypage-i18n` - マイページ翻訳
- `feat/team-i18n` - チームページ翻訳

## レビュー観点

### 1. 翻訳の品質
- 日本語が自然か
- 英語が正確か
- 用語の一貫性（同じ概念に同じ訳語）

### 2. 技術的な正確性
- `useTranslations`/`getTranslations`の正しい使用
- 翻訳キーの命名規則の一貫性
- プレースホルダーの正しい使用

### 3. 完全性
- 全てのハードコードテキストが翻訳されているか
- ja.jsonとen.jsonの両方にキーがあるか

### 4. ビルド確認
- 各ブランチでビルドが通るか
- TypeScriptエラーがないか

## レビュー手順
1. `devhive inbox`で各ワーカーの報告を確認
2. 各ブランチをチェックアウトしてコードレビュー
3. 問題があれば`devhive reply <worker> "フィードバック"`で通知
4. レビュー完了したら`devhive report "レビュー完了: <branch>"`で報告

## 完了条件
- [ ] 全ブランチのレビュー完了
- [ ] 重大な問題なし、またはフィードバック済み
