## Plane (Issue 管理)

- Plane project: `codenica/KAT` — https://plane.codenica.dev/codenica/projects/687d81ce-9021-4943-ad81-c2f6ef1fc540/issues/

- 状況確認は `plane issues KAT` または Web UI
